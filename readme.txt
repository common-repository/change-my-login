== Installation ==

1. Upload the plugin to your 'wp-content/plugins' directory, or download and install automatically through your admin panel.

2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= How to get started with "Change My Login" ? =

In your WordPress Dashboard, navigate to Settings > Permalink to get started.

= How to change login url? =

Find the "Change My Login" login section and add "Login Url" slug in text box
Example for: (Please Do not use space)
1. testlogin
2. test-login
3. test_login

Url Example: http://www.testdomain.com/testlogin

= Do I need to have coding skills to use Change My Login? =

No. Change My Login is the most beginner & user friendly custom admin login in the market.
